System Architecture

AI Job Hunter
Multi-Agent System

A complete agent blueprint to automate job discovery, applications, skill gaps & study plans

Your Current Setup

You have 2 agents in AgentiveHub 2.0: one that scrapes consultancies & job boards, and one that tailors your resume to a JD. That's a solid foundation — this architecture adds 7 more specialized agents + an orchestrator to create a fully autonomous job hunting pipeline with human approval gates.

Master Controller
🧠
Orchestrator Agent
The brain of the entire system. Receives a trigger (manual, scheduled, or event-based), decides which sub-agents to invoke, passes context between them, and maintains global job state (applied / interviewing / rejected / offer). It enforces the human approval gate — nothing gets sent without your explicit sign-off. Think of it as your personal AI chief-of-staff.

Agent 01

  • Scrapes LinkedIn, Indeed, Naukri, company career pages

Agent 02
JD Analyzer Agent
Intelligence Extractor
  • Parses JD to extract required vs nice-to-have skills
  • Identifies seniority level, team size, culture signals
  • Scores JD match against your profile (0–100)
  • Flags red flags (vague role, unrealistic expectations)
  • Categorizes: Apply Now / Study First / Skip
LangChain Claude API Structured Output

Agent 03
Resume Tailor Agent
Document Customizer
  • Takes base resume + JD → generates tailored version
  • Rewrites bullet points to mirror JD language
  • Reorders sections based on what employer values
  • ATS keyword optimization pass
  • Outputs PDF + LaTeX (you already use LaTeX!)
LangChain LaTeX Gemini API PyPDF
Agent 04
Cover Letter Agent
Persuasion Writer
  • Generates personalized cover letters per JD
  • References specific company values & recent news
  • Adapts tone: startup (casual) vs enterprise (formal)
  • 3 variants: concise / story-driven / achievement-led
  • Surfaces draft for your review before sending
LangChain Web Search Tool Claude API
Agent 05
Skill Gap Agent
Learning Strategist
  • Compares your skill profile vs JD requirements
  • Quantifies gap severity (days to close, not just yes/no)
  • Finds free resources: YouTube, docs, DeepLearning.AI
  • Outputs a prioritized study plan with timeline
  • Tracks your learning progress over time
LangChain Web Search JSON State
Agent 06
Application Tracker Agent
Pipeline Manager
  • Logs every application with date, role, company, status
  • Monitors email/inbox for replies, rejections, interview calls
  • Sends follow-up reminders if no reply in 7 days
  • Generates weekly pipeline summary report
  • Escalates hot leads to your attention immediately
Gmail MCP Notion/Airtable SQLite
Agent 07
Interview Prep Agent
Mock Interviewer
  • Generates role-specific Q&A based on JD + your resume
  • Creates system design questions for AI/ML roles
  • Simulates a full mock interview (you answer, it evaluates)
  • Identifies weak answers and suggests improvements
  • Builds a personalized STAR story bank
Claude API Multi-turn Chat Memory
Agent 08
Company Research Agent
Intel Gatherer
  • Researches company: funding, team, tech stack, culture
  • Finds Glassdoor reviews & salary ranges
  • Identifies the hiring manager on LinkedIn
  • Scans recent news (product launches, layoffs, growth)
  • Produces a 1-page company brief before each interview
Web Search LinkedIn API Tavily
Agent 09
Outreach Agent
Network Builder
  • Drafts personalized LinkedIn connection requests
  • Writes cold emails to hiring managers & recruiters
  • Schedules follow-ups in sequence (Day 1, Day 5, Day 10)
